Sponet - Plants Store (Vite + React + Tailwind) - Minimal project
===============================================================

What's included:
- React + Vite project skeleton
- Tailwind config
- Redux Toolkit cart slice (add/increase/decrease/delete)
- Pages: Landing, ProductListing, Cart
- Header with cart count
- Assets: three screenshots copied from conversation

How to run:
1. cd /workspace or wherever you download the zip and unzip
2. npm install
3. npm run dev

Note: This is a minimal starter project. Install dependencies with npm.
